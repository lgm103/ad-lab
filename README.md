Active Directory Automation Lab

Overview:
This repository contains a collection of PowerShell-based automation projects simulating real-world Active Directory (AD) administration tasks.

Each module focuses on a core operational area commonly found in enterprise environments:
-Identity lifecycle management
-Security auditing and reporting
-Policy enforcement and configuration

The goal is to demonstrate practical, job-relevant skills in automation, system administration, and infrastructure management.

Project Structure
.
├── ad-onboarding/         # User Onboarding (automated user creation)
├── jml-automation/        # Joiner-Mover-Leaver (user lifecycle automation)
├── ad-audit/              # AD auditing and compliance reporting (planned)
├── gpo-deployment/        # Group Policy automation and baseline configuration (planned)
├── shared/                # (Optional) common utilities or config templates
└── README.md              # This file

Each module contains:

Its own README with setup and usage instructions
Sample input data (where applicable)
Example outputs

Navigate into a specific folder to get started.
